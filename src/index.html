The USGS Minnesota Low-Flow web application is used as a preparation and management tool for tracking, visualization, and management of historic and future low flow measurements for the state of Minnesota.

Reference: Methods Used to Compute Low-Flow Frequency Characteristics for Continuous-Record Streamflow Stations in Minnesota, 2006

The USGS MN-Low-Flow is the public data discovery component of the Short-Term Network (STN) database. Data viewable and downloadable from this page are from the STN database.

This application relies on the STN Web Services to retrieve and display STN data. Application developers and general data seekers are encouraged to use the web services data API to retrieve STN data. Visit the STN Web Services Documentation Page for more information, including full documentation of available endpoints and examples. See the STN Data Dictionary for field names and definitions.

This application integrates with the USGS National Water Information System (NWIS) database for display of time-series water data. As a user of MN-Low-Flow you will encounter links to NWIS pages and data sources, which are maintained separately from STN and FEV.


Data Limitations

Data are provisional and are subject to revision until thoroughly reviewed and approved.

Real-time data relayed by satellite or other telemetry are automatically screened to not display improbable values until they can be verified.

Provisional data may be inaccurate due to instrument malfunctions or physical changes at the measurement site. Subsequent review based on field inspections and measurements may result in significant revisions to the data.
